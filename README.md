# 🏠 House Price Prediction using Machine Learning

## 🚀 Overview
This project predicts house prices using Machine Learning models like Random Forest and Linear Regression.

## 🧠 Models Used
- Random Forest Regressor
- Linear Regression

## ⚙️ Features
- Data preprocessing (Scaling, Encoding, Imputation)
- Pipeline implementation
- Hyperparameter tuning using GridSearchCV
- Model evaluation (MAE, MSE, RMSE, R²)
- Data visualization (Residuals & Feature Importance)

## 📊 Dataset
Synthetic dataset generated using NumPy

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run:
python house_price_model.py

## 📈 Output
- Model performance comparison
- Graphs and feature importance
